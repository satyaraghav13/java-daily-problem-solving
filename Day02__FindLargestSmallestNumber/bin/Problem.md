## Problem  1
Write a Java program to find the second largest number in an array.

## Input
int[] arr = {10, 5, 20, 8, 15};

## Output
15

## Explanation
In the given array {10, 5, 20, 8, 15}, the largest number is 20.
The second largest number is 15.






## Problem  2
Write a Java program to remove duplicate elements from an array.

## Input
int[] arr = {1, 2, 3, 2, 4, 1, 5};

## Output
1 2 3 4 5

## Explanation
In the given array, the elements 1 and 2 appear more than once.
After removing duplicates, only unique elements remain.
